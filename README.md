# Node Express

CRUD en express utilizando CouchDB como gestor de base de datos con el modulo nano 

## Instalación 

Para poder ejecutar el proyecto en necesario instalar NodeJS.



## Instalar las dependencias 
```
npm install
```
## Ejecución del proyecto  
```
npm run dep
```
